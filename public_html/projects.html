ASoIaF Character Generator

This is a simple web tool that generates characters fit to live in the world of Westeros. Users can select gender, region, and house options, and then have the website create characters based on their specifications. I used JavaScript to handle character generation as well as some jQuery to animate things nicely. Check it out here!

This Website

This is my online portfolio, which started as an assignment in ENG 100. Throughout this project I learned basic web development by coding in HTML/CSS, utilized source control for the project with Github, and managed my website hosting through FTP. I also implemented my own novel features to the website by embedding Java applets and using jQuery to create smooth page transitions.

Can One Fake Randomness?

This is a poster project, entered into the 2014 UIUC Undergraduate Research Symposium, that I worked on with my classmates Rishbabh Marya and Robert Weber, along with Professor A.J. Hildebrand. The project involved developing statistical tests in Java and JavaScript, along with heavy use of the JExcel API for data analysis. The poster was selected as a finalist for an Outstanding Presentation Award. Check it out here!
